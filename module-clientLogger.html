(inner) logger(loggerFunction) → {Client~observerCallback}

Function that can be the observer for a Client. Will call loggerFunction with a string representation of each RequestResult.

An example logging string:

Fauna POST /
Request JSON: {
  "data": ...
}
Response headers: {
   "x-faunadb-host": ...,
   "x-faunadb-build": ...,
   "connection": "close",
   "content-length": ...,
   "content-type": "application/json;charset=utf-8"
 }
Response JSON: {
   "resource": {
     "ref": { ... },
     "class": { ... },
     "ts": ...
   }
 }
Response (201): Network latency 13ms
